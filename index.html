Git c'est quoi ?

Git est un logiciel de gestion de version "distribué".
Il a été crée par Linus Torvalds l'auteur du noyau Linux.
La première version de Git fût publiée le 7 avril 2005.
Aujourd'hui (en 2016) Git est le logiciel de gestion de version le plus utilisé avec plus de 12 millions d'utilisateurs.

Il permet entre autres de sauvegardez au fil du temps les différents points de développement d'un site web ou d'un programme informatique, voir d'un simple fichier texte.

Sont fonctionnement est assez simple.
